# Crowdwise
Crowdwise helps you collect and organize feature requests to better understand customer needs and prioritize your roadmap.

### Tech Stack

- Typescript
- Next.js
- Tailwind CSS
- Zustand
- Express
- Drizzle
- PostgreSQL
- Zod
